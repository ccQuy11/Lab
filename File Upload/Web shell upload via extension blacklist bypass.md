Nội dung: trích xuất nội dung trong /home/carlos/secret

Bài này máy chủ đã chặn tất cả các file có đuôi là các file code, dựa theo nội dung gợi ý thì cần phải thêm 1 file nữa là file .htaccess

File .htaccess (viết tắt của hypertext access) là một file cấu hình được sử dụng trên web server chạy Apache, cho phép người dùng thay đổi cấu hình website ở cấp độ thư mục mà không cần phải chỉnh sửa file cấu hình chính của máy chủ.

Ví dụ của 1 file .htaccess

Dòng AddType application/x-httpd-php .abc dùng MIME (Multipurpose Internet Mail Extention) type để Apache chuyển file cho engine PHP xử lí với đuôi là .abc
